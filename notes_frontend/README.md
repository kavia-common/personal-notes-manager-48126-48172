# Notes App (Ocean Professional Theme)

A lightweight personal Notes app built with React. Create, view, edit, search, and delete notes with automatic local persistence (localStorage).

## Features
- Two-pane responsive layout (list + editor)
- Create new notes (Ctrl/Cmd+N)
- Auto-save edits with optional manual save (Ctrl/Cmd+S)
- Search by title/content, sort by last updated (desc)
- Delete with confirmation
- Offline-friendly, no backend required
- Ocean Professional theme with blue & amber accents

## Scripts
- `npm start` - run dev server at http://localhost:3000
- `npm test` - run tests
- `npm run build` - production build

## Environment
Reads any REACT_APP_* variables but does not depend on them.

## Accessibility
- Labeled inputs
- Keyboard shortcuts
- Focus ring on interactive elements
